#### 介绍
【使用的技术】<br>
数据库：MySQL 5.5<br>
数据库连接池：c3p0<br>
后端：Spring + SpringMVC + MyBatis(注解SQL)<br>
前端：Bootstrap，HTML5+CSS3+JavaScript<br>
项目管理工具：Maven<br>
其它：Tomcat 8.5

#### 使用说明

1.  使用IDEA 2017导入项目
2.  Maven -> Reimport
3.  启动Tomcat
